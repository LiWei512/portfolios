Project description

TheXchange™ is a people-driven platform that helps individuals achieve their dreams while giving their fans a vehicle to support them and share in their success. Backing people by leveraging Blockchain technology. TheXchange™ provides the vehicle to pursue their heart’s desire - wherever they live in the world if they have talent, passion and the will to fight for a purpose-filled life. TurnCoin™ is a Mega-Security Token for TheXchange™ platform. This turncoin.io website is for marketing TurnCoin™ and for managing buying TurnCoin™ orders.

Details

  • Implement the turncoin.io marketing landing page(AntD, React, Less, ReactRouter).

  • Implement the turncoin.io admin pages for users, news, events, orders, assets management(React, Redux, Redux-Saga, AntD).

  • Write rich-featured action creators and corresponding reducers.

  • Use Fetch API to call Rest APIs to fetch data and update the states in the store of Redux.

  • Involve in the design and development of Rest APIs and services to interact with a data storage layer.(Node.js, Express.js, MySQL, SequelizeORM).

  • Deploy the turncoin.io secure Rest APIs server using Nginx, Let's Encrypt and SSH on AWS EC2.

  • Use Draft.js rich text editor framework for news WYSIWYG.

  • Build several responsive email templates for order notifications.

  • Use SendGrid email service to send notification emails.

  • Work with Github and CircleCI continuous deployment environment.
